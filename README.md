自开发这个软件起, 账户的安全问题让作者焦头烂额, 自此有了三层安全层;
*	舍弃联网, 因而舍弃了绝大部分功能, 也限制了软件的拓展;
*	账户密码加密;
*	扫码登陆(v1.7.0);
